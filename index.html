Programación Orientada a Objetos: NodeJS

En este sitio web se ha implementado un sistema de información de notas académicas donde el usuario puede cargar ficheros de estudiantes para posteriormente subir ficheros con sus notas en distintas convocatorias académicas.

Usando la librería Proteic, se han desarrollado distintas gráficas para una visualización de distintas estadísticas y comparaciones.

  • Reset/Carga de ficheros
  • Consulta de elementos cargados
  • Módulo de consultas
